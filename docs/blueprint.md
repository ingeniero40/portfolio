# **App Name**: Héctor Cruz Portfolio

## Core Features:

- Project Galleries: Showcase projects in multiple galleries with navigable grids.
- Detailed Project Views: Present individual projects with detailed information and visuals.
- "About Me" Section: Introduce Héctor Cruz, highlighting his background and expertise.
- "Services Offered" Listing: Display a clear list of services provided.
- Contact Form: Provide an interactive form for potential clients to get in touch.
- Automated Portfolio Updater: AI Tool that monitors activity on linked platforms like Github, and automatically creates and displays a visually pleasing 'card' representing that activity within a 'latest work' panel on the portfolio. This feature saves the user time by removing the burden of constantly updating the portfolio.

## Style Guidelines:

- Primary color: Teal (#469990) to give a professional yet modern look.
- Background color: Dark gray (#333333) for a sleek, sophisticated base.
- Accent color: Light Green (#70db70) to provide highlights and call to action buttons.
- Body and headline font: 'Inter', a grotesque sans-serif.
- Modern, minimalist layout with a focus on clear visual hierarchy.
- Simple, professional icons to represent skills and services.
- Subtle animations on hover and transitions to enhance user experience without being distracting.
